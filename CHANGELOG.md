# 更新日志

## [1.1.2] - 2026-05-25

### 新增

- GDI 对象缓存池（GdiCache），减少控件绘图的资源开销
- 弱事件实现（WeakEvent），优化事件订阅的内存管理
- 更新检查 FetchResult 封装，区分正常/错误/无更新状态
- 应用程序图标加载功能

### 变更

- 重写应用程序启动逻辑，支持最小化启动
- 将同步更新检查改为异步实现，避免阻塞 UI 线程
- 细化网络请求异常和 API 错误的提示信息
- 过滤草稿版和预发布版本的更新检测
- 重构亮度/Gamma 控制逻辑，添加缓存减少重复调用
- 重构 GC 辅助类（GcHelper）与工作集修剪功能
- 重构设置存储序列化逻辑为线程安全处理
- 调整项目配置，优化垃圾回收行为
- 优化窗体和控件的资源释放逻辑
- 更新 README 文档与项目截图
- 整理文档格式与代码注释规范

### 修复

- 修复多处资源未正确释放的问题

### 移除

- 移除过时的 AutoApplyService 服务

## [1.1.1] - 2026-05-24

### 新增

- 系统主题监听功能，自动跟随 Windows 深色/浅色主题切换界面配色
- 单显示器独立 Gamma 预设控制，支持通过托盘菜单快速切换单屏配置
- 显示器热插拔检测与自动清理失效配置
- EDID 解析与厂商名称映射，优化显示器显示名称（MonitorManager 重构）
- 显示器位置推断逻辑，智能排序多显示器列表

### 变更

- 重构自定义控件（ModernSlider、ToggleSwitch）背景绘制逻辑，统一控件背景渲染
- 优化背景图片绘制缓存，提升界面绘制性能
- 优化亮度控制面板 UI 布局，统一控件排版

### 修复

- 修复自定义控件透明背景问题，避免默认背景覆盖导致的显示异常
- 护眼界面预设颜色按钮居中对齐
- 修正 ScheduleConfigForm 的窗体类型标识

## [1.1.0] - 2026-05-23

### 新增

- 多显示器独立 Gamma 调度配置功能，支持为每个显示器单独设置不同时段的 Gamma 预设
- 定时调度配置窗体（ScheduleConfigForm），支持多时段、多显示器独立预设配置
- PerDisplayGamma 和 ScheduleSegment 数据模型，重构用户设置调度结构
- 按显示器独立应用 Gamma 方法（ApplyGammaPerScreen）
- 预设定义类（PresetDefinitions），管理内置和自定义 Gamma 预设
- 设置迁移逻辑，兼容旧版调度配置自动升级
- 贡献指南文档（CONTRIBUTING.md）

### 变更

- 调整主窗体大小和布局，添加显示器选择和调度配置入口
- 调整设计常量 ContentWidth 适配新 UI 布局

### 移除

- 移除 README 中的 GitHub Release 徽章
